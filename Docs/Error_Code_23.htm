Information:

This type of error code occurs when the software detects that the password you have entered is "Weak" in nature & can be cracked easily.

Context:

By default it doesn't allow to put in weak passwords & if it detects that the password is weak then the vault cannot open it is done so to prevent unauthorised access to the vault by any other third party.

Cause:

It might happen due to the switching of the cofiguration. This Error occurs only on specific conditions:

When the password is too weak & when under "[Dev]" the value "password_check"  is set to '0'.

Fix:

It might be fixed by just modifing the file & settingn the value to '1'.

This feature might be used AT YOUR OWN RISK it is not recommended to do so as it might have serious privacy problems.
